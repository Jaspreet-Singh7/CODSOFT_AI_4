## 📝 Task Description
Create a recommendation system that suggests movies to users based on their preferences using Collaborative Filtering and Content-Based Filtering techniques.

---

## ✅ What I Built
A complete Hybrid Recommendation System combining 3 algorithms for accurate movie recommendations.

### 🧠 Algorithms Implemented

**1. Collaborative Filtering (User-Based)**
```
Find similar users → Get their liked movies → Recommend unseen ones
Similarity: Cosine Similarity on rating vectors
```

**2. Content-Based Filtering**
```
Build user genre profile → Match with unseen movies → Recommend best matches
Features: Genre one-hot encoding + rating boost
```

**3. Hybrid System**
```
Hybrid Score = CF(60%) + Content-Based(40%)
Best of both worlds!
```
